# SandSimulation-C
Sand Simulation programmed in C

Prerequisites:

apt install gcc libsdl2-dev libsdl2-image-dev -y

Compile:

gcc *.c -o sand -lSDL2 -lSDL2_image

Run:

./sand

Left Control adds 1 sand pixel & Right Control removes 1 sand pixel
